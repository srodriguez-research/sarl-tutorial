# SARL tutorial examples

## Software requirements

- SARL 0.15.1 <https://sarl.io>
- Ollama 0.15.6 <https://ollama.com/>

## Slides and Docs

Slides and documentation can be found in the [docs/] folder

## Running examples

Agents can be started directly for the SARL IDE or using any of the methods described in the sarl documentation.

### LLM Agents

Start ollama first

```shell
ollama serve
ollama pull llama3.2
```

Run agents in the LLM package
